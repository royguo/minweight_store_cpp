# 2026-06-07

## C++17 兼容改造

- 将项目默认 C++ 标准从 C++20 调整为 C++17：
  - 顶层 `CMakeLists.txt` 默认 `CMAKE_CXX_STANDARD=17`。
  - `src/minpatricia/CMakeLists.txt` 改为 `target_compile_features(minpatricia PUBLIC cxx_std_17)`。
- 移除 `minpatricia` 对 C++20 标准库能力的依赖：
  - 用仓库内置 `minpatricia::Span<T>` 替代 `std::span`，保留 pointer+size 的零拷贝 key view 语义。
  - 用 C++17 traits/static_assert 替代 concepts，继续约束 `RecordStore` 和 `NodeStore` 形状。
  - 用自实现 8-bit leading-zero helper 替代 `std::countl_zero`。
  - 将测试中的 `std::map::contains` 改为 C++17 兼容的 `find` 写法。
- 更新当前状态文档：
  - `README.md`
  - `docs/design/minpatricia.md`
  - `docs/design/minpatricia_port_plan.md`
  - `docs/design/minpatricia_go_ut_coverage.md`
  - `docs/trackers/resolved.md`
  - `src/minpatricia/README.md`

## 验证

- C++17 Debug/默认配置：
  - `cmake -S . -B build-cxx17 -DCMAKE_CXX_STANDARD=17 -DMINWEIGHT_BUILD_TESTS=ON -DMINWEIGHT_BUILD_BENCHMARKS=ON`
  - `cmake --build build-cxx17 -j2`
  - `ctest --test-dir build-cxx17 --output-on-failure`
  - 结果：`minpatricia_tests` 通过。
- C++17 Release 配置：
  - `cmake -S . -B build-cxx17-release -DCMAKE_CXX_STANDARD=17 -DCMAKE_BUILD_TYPE=Release -DMINWEIGHT_BUILD_TESTS=ON -DMINWEIGHT_BUILD_BENCHMARKS=ON`
  - `cmake --build build-cxx17-release -j2`
  - `ctest --test-dir build-cxx17-release --output-on-failure`
  - 结果：`minpatricia_tests` 通过，耗时约 0.54s。
